grid布局学习

发表于 2024-07-25 | 分类于 Css

grid布局

  • 父容器使用display: grid/inline-grid;
  • 一个m行n列的网格，需要使用m+1条横向，n+1条纵向网格线组成；

使用grid-template-rows、grid-template-columns这两个属性来划分行和列。

  • 重复函数

  • auto-fill 自动填充

  • auto

  • fr片段划分

  • minmax取值

网格间距

grid-row-gap设置行间距；grid-column-gap设置列间距

合并写法grid-gap: num1 num2,num1表示行间距，num2表格列间距

网格横纵向布局

grid-auto-flow:row横向布局

grid-auto-flow:column纵向布局

项目对齐方式

justify-items水平对齐方式，值有start|center|end

align-items垂直对齐方式，值有start|center|end

复合写法：

place-items: align-items(垂直方向) justify-items(水平方向)
