<code>双飞翼布局和圣杯布局就是边两栏宽度固定，中间栏宽度自适应，并且保证中间部分优先加载。</code>

区别：双飞翼布局和圣杯布局所要实现的最终效果都是一样的，只是实现的方法不同而已。<font color="yellowgreen">双飞翼布局是通过float与margin负值配合实现的。</font><font color="pink">圣杯布局是通过float、padding、(相对定位)position:relative; 配合实现的。</font>

> ### 双飞翼布局

<font color="lightblue">侧边两栏宽度固定，中间栏宽度自适应</font>

双飞翼布局 Demo:

```
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }
        body {
            min-width: 520px;  /* 为了保证窗口不能缩太小无法展示左右，可以给body加上 min-width */
        }
        .main {
            width: 100%;
            height: 360px;
            background-color: #ccc;
            float: left
        }
        .left {
            width: 200px;
            height: 200px;
            background-color: gold;
            float: left;
            margin-left: -100%;  /* 双飞翼布局的关键代码 */
        }
        .right {
            width: 200px;
            height: 200px;
            background-color: pink;
            float: left;
            margin-left: -200px;  /* 双飞翼布局的关键代码 */
        }
        .middle {
            height: 360px;
            background-color: lightblue;
            margin: 0 210px;  /* 双飞翼布局的关键代码 */
        }
    </style>
</head>
    
<body>
    <!-- 双飞翼布局有左中右三块，把中间块放最上面，是为了用户体检好，html代码是从上到下加载的，上面的先加载，在网络环境较差的情况下，保证中间区域先加载出来 -->

    <div class="main">
        <div class="middle"></div>
    </div>
    <div class="left"></div>
    <div class="right"></div>
</body>

</html>

```

> ### 反（双飞翼布局）

<font color="lightblue">两边自适应大小，中间部分固定不变</font>

需要flex布局的flex:1和max-width或者min-width配合

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
        /*实现效果：两边自适应大小，中间部分固定不变*/
        /*需要flex布局的flex:1和max-width或者min-width配合*/
        * {
            margin: 0;
            padding: 0;
        }
        ul,li {
            list-style-type: none;
        }
        ul {
            display: flex;
        }
        li {
            flex: 1;
            border: 1px solid #000;
        }
        li.first {
            height: 300px;
            background: gold;
        }
        li.second {
            min-width: 500px;
            max-width: 500px;
            background: lightblue;
            height: 500px;
        }
        li.third {
            height: 300px;
            background: pink;
        }
    </style>
</head>
<body>
    <ul>
        <li class="first">1</li>
        <li class="second">2</li>
        <li class="third">3</li>
    </ul>
</body>
</html>

```

> ### 圣杯布局

原理上面已经说过了。直接上demo:

```
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>圣杯布局</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }
        body {
            /* 这里的最小宽度不要小于650px,否则会换行，可能给padding和left的值有关 */
            min-width: 700px;  /* 为了保证窗口不能缩太小无法展示左右，可以给body加上 min-width */
        }
        .container {
            padding: 0 210px;  /* 圣杯布局的关键代码*/
            overflow: hidden;
            background-color: #ccc;
        }
        .left, .right, .middle {
            position: relative;  /* 圣杯布局的关键代码*/
        }
        .left {
            width: 200px;
            height: 200px;
            background-color: gold;
            float: left;
            margin-left: -100%;  /* 圣杯布局的关键代码*/
            left: -210px;  /* 圣杯布局的关键代码*/
        }
        .right {
            width: 200px;
            height: 200px;
            background-color: pink;
            float: left;
            margin-left: -200px;  /* 圣杯布局的关键代码*/
            right: -210px;  /* 圣杯布局的关键代码*/
        }
        .middle {
            width: 100%;
            height: 360px;
            float: left;
            background-color: lightblue;
        }
    </style>
</head>
    
<body>
    <div class="container">
        <div class="middle"></div>
        <div class="left"></div>
        <div class="right"></div>
    </div>
</body>

</html>

```
