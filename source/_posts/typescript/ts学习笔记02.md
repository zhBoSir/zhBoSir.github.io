> ## <code>1.keyof 与 object.keys</code>

keyof 与 Object.keys 略有相似，只不过 keyof 取 interface 的键

```js
interface Point {
  x: number;
  y: number;
}

type keys = keyof Point;  // type keys = "x" | "y"
```
