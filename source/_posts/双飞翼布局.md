> ### 双飞翼布局

<font color="lightblue">侧边两栏宽度固定，中间栏宽度自适应</font>

双飞翼布局 Demo:

```
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }
        .main {
            width: 100%;
            height: 360px;
            background-color: #ccc;
            float: left
        }
        .left {
            width: 200px;
            height: 200px;
            background-color: gold;
            float: left;
            margin-left: -100%;  /* 双飞翼布局的关键代码 */
        }
        .right {
            width: 200px;
            height: 200px;
            background-color: pink;
            float: left;
            margin-left: -200px;  /* 双飞翼布局的关键代码 */
        }
        .middle {
            height: 360px;
            background-color: lightblue;
            margin: 0 210px;  /* 双飞翼布局的关键代码 */
        }
    </style>
</head>
    
<body>
    <div class="main">
        <div class="middle"></div>
    </div>
    <div class="left"></div>
    <div class="right"></div>
</body>

</html>

```
